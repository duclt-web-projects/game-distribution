# Project

This is a FE web game project

# Requirements

This project requires

- Nodejs (>= v.16)
- npm or yarn

# Installation

1. Clone project https://github.com/duclt-web-projects/game-distribution
2. Install packages

```
npm install
```

3. Copy `.env` from `.env.example` and setup Environment Parameters
4. Run project

```
npm run dev
```
